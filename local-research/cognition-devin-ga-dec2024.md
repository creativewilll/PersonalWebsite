### Devin by Cognition: General Availability Launch (December 2024)

**Launch Date and Availability**  
Devin became **generally available on December 10, 2024**, as announced in Cognition's official blog post. It's now accessible to all engineering teams via [app.devin.ai](https://app.devin.ai). Previously in private preview, it's positioned as the "world's first AI software engineer" capable of handling end-to-end engineering tasks autonomously.

**Pricing**  
- Starts at **$500 per month** for engineering teams.  
- Includes:  
  - Unlimited seats (team-focused).  
  - Access to Devin’s API, Slack integration, and IDE extension (beta).  
  - Onboarding session and direct support from Cognition's engineering team.  
This structure supports enterprise-scale use without per-user limits.

**Key Capabilities**  
Devin acts as an autonomous AI agent for software engineering, going beyond code completion tools like GitHub Copilot. It plans, executes, debugs, and reports on tasks using a full development environment (command line, code editor, browser). Highlights include:  
- **Code Tasks**: Fix frontend bugs/edge cases, create first-draft PRs for backlog items, refactors, unit/E2E testing, build SaaS integrations.  
- **Collaboration Features**: Auto-wakes on PR comments/lint failures, PR previews, links sessions to PRs.  
- **Repo Awareness**: Automatically scans repositories to generate "Repo Knowledge" for faster work in existing codebases.  
- **IDE Integration** (Beta): Cmd+G shortcut in VS Code/Cursor; hand off async tasks, review/accept code locally.  
- **Behavior Controls**: Adjustable "Agency" mode (proposes plans for approval on complex tasks).  
- **Non-Code Tasks**: Web research, repetitive automation, online bookings/reservations, backoffice workflows.  
- **Process**: Users give natural language prompts; Devin plans step-by-step, searches the web for info, codes/tests/debugs, and reports progress in real-time. Humans can intervene via chat.  

Example: Built a Pong game and full website from scratch in <20 minutes (early demos).

**Performance Improvements (Dec 2024 Updates)**  
- ~10% faster and more cost-efficient (fewer Agent Compute Units/ACUs needed).  
- Time to first action: ~10s (from 25s).  
- Snapshot creation: ~15s (from 30min).  
- Fixes for crashing/stuck sessions; refunds available for issues.  

**Enterprise Features**  
- SSO (Okta), auto-join for company domains, custom onboarding, usage insights/alerts.

**Benchmarks and Limitations**  
No standardized public benchmarks (e.g., SWE-Bench scores) in search results, but Cognition emphasizes real-world utility over perfection:  
- "Makes engineers 10x more productive" (vision statement).  
- Strong on complete projects vs. snippets, but "often makes mistakes, writes bugs, or gets stuck" – requires human oversight.  
- Compared to Copilot: More autonomous (full jobs vs. suggestions/reviews).  

**Context and Competition**  
Launched by Cognition (valued at $2B post-funding), Devin targets teams delegating routine work to focus on creative tasks. Competes with AI agents like Anthropic's Computer Use, OpenAI Operator, Google's Jarvis. Early access showed it handling complex tasks faster than humans in demos.
