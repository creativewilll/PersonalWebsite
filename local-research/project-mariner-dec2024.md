### Overview of Google Project Mariner
Google Project Mariner was an experimental AI agent developed by Google DeepMind, unveiled in **December 2024** as a prototype for **autonomous web navigation**. It operated via a Chrome extension, using the Gemini model to mimic human-like interactions with websites. The agent captured browser screenshots, processed them in the cloud for multimodal reasoning (interpreting text, images, forms, and code), planned multi-step actions, and executed tasks like cursor movements, clicking, and form-filling.

Key capabilities demonstrated:
- **Task automation**: Research, planning, data entry, shopping cart creation (e.g., on Instacart), job searches (e.g., on Climatebase), hiring Taskers (e.g., on TaskRabbit), and ordering ingredients.
- **Performance**: Achieved 83.5% success on the WebVoyager benchmark.
- **UX features**: Ran in the active Chrome tab with user oversight; users could intervene, prompt, or stop it. It learned workflows for future reuse.

**Built-in limitations** (for safety): Could not complete purchases, accept cookies, agree to terms of service, or handle certain interactive elements like Google Search.

### Availability and Evolution
- **Initial rollout**: Available to select testers in Dec 2024; expanded to Google AI Ultra subscribers in the US by May 2025.
- **Integration plans**: Capabilities were slated for Gemini API, Vertex AI, and Google Search's AI Mode.
- **Demos and reception**: Showcased at Google I/O 2025; praised as a "new UX paradigm" but noted delays (e.g., 5-second action pauses).

### Shutdown
Project Mariner was **quietly discontinued** around **March-May 2026**, with official shutdown on **May 4, 2026** (just days before today). Reasons appear tied to shifting priorities in the "AI agent race." The landing page now confirms it's defunct, and staff were reassigned.

**Legacy**: Core tech (browser control via screenshots/visual recognition) lives on in **Gemini API** and **Gemini Agent**, enabling developers to build similar tools for web tasks like form-filling and navigation.
