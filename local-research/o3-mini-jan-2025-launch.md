# OpenAI o3-mini Launch Research - January 31, 2025

## Launch Overview
- **Date**: January 31, 2025
- **Product**: o3-mini and o3-mini-high
- **Positioning**: Cost-efficient reasoning model succeeding o1-mini
- **Key Claim**: 95% cost reduction per token since GPT-4

## Reasoning Effort Levels
| Level | Availability | Performance |
|-------|-------------|-------------|
| Low | API only | Comparable to o1-mini, lower cost/latency |
| Medium | All ChatGPT tiers + API | Matches o1 performance |
| High | Plus/Team/Pro + API | Outperforms o1 and o1-mini |

## ChatGPT Tier Access
| Tier | Access | Rate Limits |
|------|--------|-------------|
| Free | o3-mini (medium effort via "Reason" button) | Limited |
| Plus/Team | o3-mini + o3-mini-high | 150 messages/day (up from 50 for o1-mini) |
| Pro | Unlimited o3-mini + o3-mini-high | Unlimited |
| Enterprise/Edu | Rolling out shortly | N/A |

## Key Features
- STEM domains focus (science, math, coding)
- Function calling support
- Structured Outputs
- Developer messages
- Streaming
- Search with web links
- NO vision support

## Competitive Context
- Positioned as rival to DeepSeek models
- First reasoning model available to free tier users
- Replaces o1-mini in ChatGPT model picker
