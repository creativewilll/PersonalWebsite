# AlphaFold 3: DeepMind's Revolutionary Protein Structure Prediction System

## Publication & Launch Overview

**AlphaFold 3** was unveiled by **Google DeepMind and Isomorphic Labs** on **May 8, 2024**, with findings published in the prestigious journal *Nature*. This represents a major evolutionary leap beyond AlphaFold 2, significantly expanding the model's capabilities.

## Key Capabilities

AlphaFold 3 can predict the structure and interactions of:
- **Proteins**
- **DNA and RNA**
- **Small molecules (ligands)** — including drug-like compounds
- **Chemical modifications** that control cellular function
- **Protein-antibody complexes**

The system operates as a unified deep-learning framework, computing entire molecular complexes holistically in a single model.

## Prediction Accuracy Benchmarks

### Protein-Ligand Interactions
- **76% accuracy** for protein interactions with small molecules
- Compared to 52% for previous best predictive software
- **50% more accurate** than best traditional methods on PoseBusters benchmark

### DNA Interactions
- **65% accuracy** for DNA interactions
- Compared to 28% for the next leading system

### Protein-Protein Interactions
- **62% accuracy**
- **More than doubled** AlphaFold 2's performance

### Antibody-Antigen Prediction
- Substantially higher accuracy compared to AlphaFold-Multimer v.2.3

## Free Server & Accessibility

**AlphaFold Server** launched as a **free platform** for non-commercial research, allowing scientists worldwide to:
- Model molecular structures with a few clicks
- Access the world's most accurate protein interaction prediction tool

## Code Availability Update

- **November 2024**: AlphaFold 3 model code and weights were released for academic use
- **Initial limitation**: The May 2024 publication faced criticism for lacking open-source code, restricting reproducibility and commercial applications
