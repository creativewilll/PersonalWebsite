# Google Gemini 2.0 Flash Experimental: December 11, 2024 Announcement

## Overview
Google announced **Gemini 2.0 Flash Experimental** on December 11, 2024, positioning it as a foundational AI model designed for the "agentic era" with significant improvements over previous generations.

## Key Specifications

### Performance
- **2x faster** than Gemini 1.5 Pro on certain benchmarks
- **Outperforms Gemini 1.5 Pro** on key benchmarks while maintaining comparable speed to 1.5 Flash
- Displaces 1.5 Pro as the flagship Gemini model
- Enhanced in coding, image analysis, math skills, and "factuality"

### Context & Processing
- **1 million token context window**
- Bidirectional streaming with Live API
- Multi-turn reasoning capabilities

## Multimodal Capabilities

### Inputs
- Text
- Images (up to 3,000 images per prompt; 7 MB max file size for inline data, 30 MB from Cloud Storage)
- Video (up to 10 videos; ~45 minutes with audio, ~1 hour without)
- Audio

### Outputs
- **Text** (available to all developers)
- **Native Image Generation** (early access partners)
- **Steerable Text-to-Speech Audio** (early access partners)
  - 8 customizable voices optimized for different accents and languages
  - Controllable speed and tone (e.g., "speak like a pirate")
  - Multilingual support with watermarking

## Advanced Features

- **Native Tool Use**: Built-in integration with Google Search, Maps, and code execution
- **Function Calling**: Compositional function-calling and third-party API integration
- **Agentic Capabilities**: Native UI action capabilities for autonomous agent experiences
- **Complex Processing**: Enhanced multimodal reasoning, long context understanding, and complex instruction following

## Availability Timeline

| Release Phase | Date | Access |
|---|---|---|
| **Experimental Preview** | December 11, 2024 | Gemini API (Google AI Studio, Vertex AI); Gemini web chat dropdown; Limited early access partners for audio/image generation |
| **General Availability** | January 2025 | Wider rollout with additional model variants |
