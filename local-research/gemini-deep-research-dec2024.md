## Launch & Current Status

**Deep Research** debuted in **December 2024** as Google's first agentic AI feature in Gemini. It has since evolved significantly:

- **Initial Launch (Dec 2024)**: Powered by Gemini 1.5 Pro, available to Gemini Advanced subscribers
- **Current Evolution (2025-2026)**: Now powered by Gemini 3.1 Pro and Gemini 2.0 Flash Thinking, with expanded capabilities
- **Availability**: Over 45 languages in 150+ countries; now available via Gemini API for developers

## How It Works

The feature operates as an autonomous research agent with a multi-step process:

1. **Planning**: Creates a collaborative research plan for user review and refinement
2. **Execution**: Autonomously searches up to hundreds of websites
3. **Iteration**: Continuously refines analysis based on findings
4. **Synthesis**: Generates comprehensive, cited multi-page reports within minutes

## Key Capabilities

- **Multi-source research** across the open web and custom data sources
- **Native visualizations**: Charts, infographics, and graphs (new feature)
- **Integration tools**: Google Search, MCP servers, Code Execution, File Search, Gmail/Drive/Chat access
- **Extended context**: Handles longer, more complex research queries
- **Export functionality**: Direct export to Google Docs
